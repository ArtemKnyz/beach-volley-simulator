# 🏐 Beach Volley Tournament Simulator
Симулятор турнира по пляжному волейболу (микст 2x2).

- 3 мужчины и 3 девушки
- Формируются все возможные команды (м+ж)
- Генерируются все уникальные матчи без повторений (18 игр)
- Симулируются результаты матчей
- Рассчитывается рейтинг игроков

  ## ⚙️ Логика турнира

- Всего команд: 9 (3x3 комбинации)
- Матчи: 18 уникальных (без пересечения игроков)
- Победитель определяется вероятностной моделью (skill игроков)
- Игра ведётся до выбранного количества очков (11, 15, 21, 25)

  ## 📊 Пример результатов

M1+F1 vs M2+F2 21:17  
M1+F2 vs M3+F1 19:21  

Рейтинг игроков:
M1 W:8 L:4 Rating:0.72  
F1 W:7 L:5 Rating:0.68  

## 🛠️ Технологии

- Java 17+
- OOP
- Streams API
- Random simulation

  ## 🧱 Архитектура

- Player — игрок и статистика
- Team — команда (record)
- Match — матч и логика игры
- GameMode — режим игры (enum)
- TournamentService — генерация турнира
